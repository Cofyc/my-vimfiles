# Spell Check

http://vimdoc.sourceforge.net/htmldoc/spell.html
http://robots.thoughtbot.com/vim-spell-checking

## Usage

```
:set spell
:set nospell
```

## Additional words

file: ~/.vim/spell/en.utf-8.add

#### Add one word

type `zg` on word

#### Update `spellfile` manually

```
:mkspell! %
```
