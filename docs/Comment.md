# Comment

## Basic

```
:<leader>c<space> # toggle
:<leader>cc       # commented by line marker
:<leader>cn       # commented by nested marker (if possible)
:<leader>cm       # commented by one set of nested marker
:<leader>cl       # left aligned
:<leader>cu       # uncomment
```
