# Spelling

## Shortcuts

### Turn spelling check on and off

```
:set spell    # on
:set nospell  # off
```

### Add word into additional word list

Type `zg` on word

### Update `spellfile` manually

```
:mkspell! %
```

### Show alternatives on word

```
z=
```
